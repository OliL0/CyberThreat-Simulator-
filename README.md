# CyberThreat-Simulator-
An interactive education project designed to teach cyber security awareness through simulated scenarios and quizzes

Project Overview 
Cyber Threat simulator is an education tool that is built to help recognise and respond to common cybersecurity threats.
It includes interactive phishing and social engineering quizzes that have real world inspired scenarios making the learning more interactive and hands on

Features
- Phishing Quiz - Learn to identify fake emails, suspicious links
- Social Engineering Quiz - Explore simulated real world situations including call transcripts, chat messages. That teach users how attackers manipulate trust, authority and urgency to gain information
- Instant feedback - After completing each quiz scenario you get an explanation on why it is phishing or social engineering and results at the end of the quiz for a total score
- Educational Focus - Perfect for schools, training programs and workshops

Technologies Used 
- Frontend: HTML / CSS / JS 
- Data Storage: JSON 

Educational Purpose 
This project is designed only for training and awareness not for any malicious use 
This promotes safe online practices and helps users be able to identify and avoid cyber threats in a controlled environment.
